# Payday Infrastructure

Terraform code to provision:
- AWS VPC
- EKS Kubernetes Cluster
- Remote Terraform state (S3)

## Setup

1. Install Terraform & AWS CLI
2. Configure AWS credentials
3. Run:

terraform init
terraform plan
terraform validate
terraform apply
